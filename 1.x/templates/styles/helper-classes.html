• To make text less prominent, change its color using the .subdued class. This change will still persist if the class is within a link.
  • You can mark content as full width. For example, styling a full width button by using the .full-width class. This class requires 'display' to be styled as inline-block or block.
  • Hidden content
    • To hide an element but not remove it from the page spacing, use the .hidden class.
    • You can hide content visually and from the page spacing (but can still be read by screen readers) using the .visually-hidden class.
  • Clearing floats
    • To clear child elements using clearfix, you can use the .clear class. Learn more about the hack .
    • To clear previous sibling floats, you can use the .clear-left and .clear-right classes.
