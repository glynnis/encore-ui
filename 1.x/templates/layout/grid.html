Encore UI includes a grid system forked from Angular Material's layout module with minor usability enhancements to provide an assortment of attribute-based layout options based on the flexbox layout model. Included are intuitive attribute based styles that ease the creation of responsive row and/or column based page layouts.

Angular Material's responsive CSS layout is built on flexbox. The layout system is based upon element attributes rather than CSS classes. Attributes provide an easy way to set a value (eg layout="row") and help separate concerns: attributes define layout, and classes define styling.

We have a reference to the old Pure CSS grid for your convenience, but note that the Pure CSS grid is slated for deprecation and shouldn't be used. Any pages using it should be updated.
