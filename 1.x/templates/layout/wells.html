Page Wells

To create a well-effect, you can use the .well class — a box with significant padding and light gray background. You might use Page wells to communicate why data is not present for the user, usually if data can only be seen with permissions the user has not been granted. In cases where a user is able to add data without a permissions change, a button can be displayed to prompt the user to add data.

Modal Wells

Typically you can use Modal wells in the context of modals, to provide a summary of data entered or decisions made by the user in a form, to provide a clearer picture to the user of the changes they are about to submit.
