rxFormItem
directive in module rxForm

Deprecated API

Description

DEPRECATED - This directive has been marked as deprecated and will be removed in a future release of the EncoreUI framework. Please see current rxForm documentation for updated functionality.

This directive is used to wrap input fields and select boxes in the proper HTML. It will attach the <label> to a transcluded input using the id attribute of the input. If no id attribute exists, it will create one.

Usage

as element:
<rx-form-item
       label="{String}"
       prefix="{String}"
       suffix="{String}"
       description="{String}">
</rx-form-item>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
labelString
  • Text to use for
prefixString
  • Text to include to the left of content
suffixString
  • Text to include to the right of content
descriptionString
  • Text to place below input
