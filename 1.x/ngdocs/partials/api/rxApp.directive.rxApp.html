rxApp
directive in module rxApp

Description

Responsible for creating the HTML necessary for a common Encore layout.

Usage

as element:
<rx-app
       [site-title]="{string}"
       [menu]="{array}"
       [collapsible-nav]="{string}"
       [collapsed-nav]="{string}"
       [new-instance]="{boolean}"
       [hide-feeback]="{boolean}"
       [logout-url]="{string}">
</rx-app>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
[siteTitle]string

Title of site to use in upper right hand corner

[menu]array

Menu items used for left-hand navigation

[collapsibleNav]string

Set to 'true' if the navigation menu should be collapsible

[collapsedNav]string

Binding for the collapsed state of the menu.

[newInstance]boolean

Whether the menu items should be a new instance of rxAppRoutes

[hideFeeback]boolean

Whether to hide the 'feedback' link or not (defaults to show it)

[logoutUrl]string

URL to pass to rx-logout

Example

<rx-app site-title="Custom Title"></rx-app>
