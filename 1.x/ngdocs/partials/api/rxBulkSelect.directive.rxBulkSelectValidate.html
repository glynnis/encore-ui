rxBulkSelectValidate
directive in module rxBulkSelect

Description

A directive used to validate rxBulkSelect in a form. The directive should be placed on the same element as rxBulkSelect. The form will be invalid when no items are selected and valid when at least one item is selected.

Dependencies

  • rxBulkSelect

Usage

as attribute
<ANY rx-bulk-select-validate>
   ...
</ANY>
