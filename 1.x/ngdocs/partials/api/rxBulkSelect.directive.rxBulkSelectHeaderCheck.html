rxBulkSelectHeaderCheck
directive in module rxBulkSelect

Description

A directive you place on your <th> element representing the checkbox column. This places a checkbox in the header, which will select all items on the current page when clicked.

Dependencies

  • rxBulkSelect

Usage

as attribute
<ANY rx-bulk-select-header-check>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Example

<th rx-bulk-select-header-check></th>
