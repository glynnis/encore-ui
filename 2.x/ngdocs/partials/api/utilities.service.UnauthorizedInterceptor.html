UnauthorizedInterceptor
service in module utilities

Description

Simple injector which will intercept HTTP responses. If a HTTP 401 response error code is returned, the ui redirects to /login.

Dependencies

  • $q
  • @window
  • Session

Example

angular.module('encoreApp', ['encore.ui'])
    .config(function ($httpProvider) {
        $httpProvider.interceptors.push('UnauthorizedInterceptor');
    });
